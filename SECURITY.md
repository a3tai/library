# Security Policy

## Supported Versions

Security fixes target the latest released version of A3T: Library.

## Reporting a Vulnerability

Please report vulnerabilities privately through GitHub Security Advisories for this repository. Do not open a public issue for suspected security vulnerabilities.

Include:

- Affected version or commit.
- Operating system.
- Steps to reproduce.
- Impact and any known workaround.

We will acknowledge valid reports as soon as practical and coordinate fixes before public disclosure.
