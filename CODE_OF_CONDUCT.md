# Code of Conduct

This project is intended to be a practical, respectful place to collaborate.

## Expected Behavior

- Be direct, kind, and specific.
- Assume good intent while still asking for clarity when something is ambiguous.
- Keep reviews focused on the work: correctness, maintainability, usability, security, and tests.
- Credit ideas and contributions from others.

## Unacceptable Behavior

- Harassment, threats, insults, or sexualized language.
- Personal attacks or sustained off-topic disruption.
- Publishing private information without explicit permission.
- Intentionally misleading maintainers or contributors.

## Reporting

Report conduct issues privately through GitHub repository maintainers. Security vulnerabilities should follow [SECURITY.md](SECURITY.md).

Maintainers may remove comments, close issues, block accounts, or take other reasonable action to keep the project usable for contributors.
